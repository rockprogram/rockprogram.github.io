July 6, 2021

工欲善其事，必先利其器，这里专门记录各种工程搭建经验、工具使用技巧。古代剑客对战，近代牛仔决斗，拼的都是谁出手快，想要出手快，就要提前做好各种准备。这就是我写本篇文章的初衷。

Java

  1. maven配置镜像，编辑settings.xml，

    <mirrors>
    <mirror>
        <id>alibaba</id>
        <mirrorOf>central</mirrorOf>
        <name>aliyun maven</name>
        <url>http://maven.aliyun.com/nexus/content/groups/public/</url>
    </mirror>
</mirrors>

Python

  1. pip下载速度慢，需要一个国内镜像源，编辑~/.pip/pip.conf

    [global]
index-url = http://mirrors.aliyun.com/pypi/simple/

[install]
trusted-host = mirrors.aliyun.com

  2. supervisor，强大的应用运行时管理工具

    1. 程序配置demo，修改/etc/supervisor/conf.d/tornado.conf：

      [program:tornado]
directory=/opt/tornado
command=python /opt/tornado/start.py
user=guanyu
autorestart=true
autostart=true

    2. 命令：

      1. sudo apt-get install supervisor
      2. sudo service supervisor start/stop/restart
      3. supervisorctl start/stop tornado

    3. 如果启动报错：Permission denied: file: /usr/local/lib/python2.7/socket.py，修改/etc/supervisor/supervisord.conf：

      [unix_http_server]
file=/tmp/supervisor.sock   ; 保持不变(the path to the socket file)
chmod=0766                 ; 修改权限 socket file mode (default 0700)

Linux

  1. 命令行排查问题

    1. AWK
      1. 求和：awk ‘{sum += $1};END {print sum}’ file
      2. 指定分隔符：awk -F ‘char’
    2. sort
      1. 指定分隔符：-t ‘char’
      2. 指定按列排序：-k n
      3. 默认按字母顺序排序，如果要按数字排序：-n
      4. 默认升序排序，如果需要降序排序：-r
    3. uniq
      1. uniq通常和sort合用，必须保证重复行是相邻的。
      2. 忽略大小写字母：-i
      3. 计数：-c
    4. du
      1. -m 以MB为单位输出
      2. –max-depth=1 * 当前目录所有文件

  2. Ubuntu修改时区

    1. tzselect
    2. sudo cp /usr/share/zoneinfo/Asia/Shanghai /etc/localtime

数据库

  1. UTF8字符配置，修改/etc/mysql/mysql.conf.d/mysqld.cnf：

    [client]
default-character-set=utf8mb4

[mysqld]
character-set-server = utf8mb4
collation-server = utf8mb4_unicode_ci
init_connect='SET NAMES utf8mb4'
skip-character-set-client-handshake = true

[mysql]
default-character-set = utf8mb4

Fish shell

  1. 配置config.fish：

    set fish_greeting 'Talk is cheap. Show me the code.'
set -g fish_user_paths "/usr/local/opt/mysql@5.7/bin" $fish_user_paths

  2. 配置fish_prompt.fish:

    1. Git版

      function fish_prompt --description 'Write out the prompt'
    # 快捷键映射
    bind K accept-autosuggestion

    #prompt
    set -l home_escaped (echo -n $HOME | sed 's/\//\\\\\//g')
    set -l pwd (echo -n $PWD | sed "s/^$home_escaped/~/" | sed 's/ /%20/g')
    set -l prompt_symbol ''
    switch $USER
        case root toor
            set prompt_symbol '#'
        case '*'
            set prompt_symbol '$'
    end

    # show git branch
    if not set -q __git_cb
        set __git_cb ":"(set_color brown)(git branch ^/dev/null | grep \* | sed 's/* //')(set_color normal)""
    end

    printf "[%s@%s %s%s%s]%s%s" $USER (prompt_hostname) (set_color $fish_color_cwd) $pwd (set_color normal) $__git_cb $prompt_symbol
end

    2. 标准版

      function fish_prompt --description 'Write out the prompt'
    # 快捷键映射
    bind K accept-autosuggestion
    set -l home_escaped (echo -n $HOME | sed 's/\//\\\\\//g')
    set -l pwd (echo -n $PWD | sed "s/^$home_escaped/~/" | sed 's/ /%20/g')
    set -l prompt_symbol ''
    switch "$USER"
        case root toor
            set prompt_symbol '#'
        case '*'
            set prompt_symbol '$'
    end
    printf "[%s@%s %s%s%s]%s " $USER (prompt_hostname) (set_color $fish_color_cwd) $pwd (set_color normal) $prompt_symbol
end
