要你命3000

工欲善其事，必先利其器，这里专门记录各种工程搭建经验、工具使用技巧。古代剑客对战，近代牛仔决斗，拼的都是谁出手快，想要出手快，就要提前做好各种准备。这就是我写本篇文章的初衷。 Java maven配置镜像，编辑settings.xml， <mirrors> <mirror> <id>alibaba</id> <mirrorOf>central</mirrorOf> <name>aliyun maven</name> <url>http://maven.aliyun.com/nexus/content/groups/public/</url> </mirror> </mirrors> Python pip下载速度慢，需要一个国内镜像源，编辑~/.pip/pip.conf [global] index-url = http://mirrors.aliyun.com/pypi/simple/ [install] trusted-host = mirrors.aliyun.com supervisor，强大的应用运行时管理工具 程序配置demo，修改/etc/supervisor/conf.d/tornado.conf： [program:tornado] directory=/opt/tornado command=python /opt/tornado/start.py user=guanyu autorestart=true autostart=true 命令： sudo apt-get install supervisor sudo service supervisor start/stop/restart supervisorctl start/stop tornado 如果启动报错：Permission denied: file: /usr/local/lib/python2.7/socket.py，修改/etc/supervisor/supervisord.conf： [unix_http_server] file=/tmp/supervisor.sock ; 保持不变(the path to the socket file) chmod=0766 ; 修改权限 socket file mode (default 0700) HEXO 常用命令 hexo new [title] 新建文章 hexo generate 生成静态文件 hexo server 启动服务器 hexo deploy 部署网站 HUGO 新建文章：hugo new posts/my-first-post.
